Já é possível viajar de férias para Marte!

Empresa americana tem a viagem que você sempre sonhou! Férias no planeta Marte, e a preços Escepcionais.

Fonte: Agência de Viagens Espaciais

Já é possível fazer as suas férias em Marte. É a viagem que você sempre sonhou, e que agora, está no seu alcance! A viagem é longa, mas é uma expeiencia unica. Pode levar vários acompanhantes e realizar finalmente o seu sonho!

Mas muita atenção: são necessários alguns items preparatórios antes de seguir viagem.

O que necessito para viajar?

  • Exames médicos gerais
  • Duas semanas de treino especializado
  • Passaporte interstelar válido
  • Asssinar um termo de responsabilidade

Procedimentos antes de viajar

  1. Escolher a data da viagem
  2. Escolher o número de acompanhantes
  3. Escolher o tipo de alojamento
  4. Escolher o tipo de transporte
  5. Escolher o tipo de alimentação
  6. Escolher o tipo de atividades
